# Vehicle Speed Detection using YOLOv8

This project detects vehicles and calculates speed in real time using:
- Ultralytics YOLOv8
- OpenCV
- Python 3.x

## Features
- Real-time vehicle detection
- Speed calculation
- Overspeed warnings

## How to Run
```
pip install -r requirements.txt
python speed.py
```
